1. git init (сообщаю Гиту чтобы он следил за этой папкой).
2. git config --local user.name "Artem" (автор)
    git config --local user.email example@gmail.com (email автора)

3. git status - позволяет просмотреть контрольную точку.
4. git add -A (гит будет следить за всеми файлами).
5. git commit -a -m"какие-то описания изминений на этой контрольной точке".
6. git log (информация о том, кто-что изменил).

Закидываю всё на GitHub
1. создаю новый репозиторий кнопкой "new", даю ему имя и жму "create repository".
2. копирую команду "git remote add origin https://..." и вставляю в терминал который в "vs code".
3. git push -u origin master/main (закидывает на Гитхаб файлы проекта).
